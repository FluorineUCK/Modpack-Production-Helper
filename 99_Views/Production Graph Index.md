# Production Graph Index

Use Obsidian Graph View to see the global production network. Recipe notes are first-class nodes between input items, machines, and output items, so the graph shows production steps instead of only item-to-item shortcuts.

Use Local Graph on an item note to inspect nearby recipes. Use the `Produced By` and `Used In` Dataview sections on item notes for directional upstream and downstream tables.

Recipe notes include Mermaid flowcharts for the local process view. Use Canvas for manual high-level maps such as starter ore processing, mid-game circuits, or endgame production loops.

Useful Graph View filters:

```text
tag:item
tag:recipe
tag:machine
tag:mod/minecraft
tag:status/check
```
